# Renix static website

Sito statico pronto per Netlify.

## Cosa sostituire per renderlo identico al Figma

In `/assets` sostituisci:
- `logo-renix.svg`
- `logo-renix-footer.svg`
- `hero-solar.jpg`
- `project-01.jpg`
- `icon-sustainability.svg`
- `icon-maintenance.svg`
- `icon-technology.svg`

I file `reference-1.png` ... `reference-5.png` sono solo screenshot di riferimento e non vengono usati nel sito.

## Deploy Netlify

1. Entra su Netlify
2. Add new site > Deploy manually
3. Trascina la cartella oppure lo zip
4. Collega il dominio da Domain management
